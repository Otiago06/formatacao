Roteiro de Formatação de Computador

Check list de chegada

1

Analisar condições do computador

Verificar se tudo está funcionando, quais dispositivos acompanham o equipamento e abrir Ordem de Serviço.

2

Analisar condições do sistema operacional

Analisar os softwares instalados como o softwares da RFB, Microsoft Outlook e outros gerenciadores de E-Mail ou softwares que precisem de um backup específico.

3

Analise e verificação de vírus e malwares

Analisar vírus com o KVRT (Kaspersky Virus Removal Tool) e Malwares com o Malwarebytes AdwCleaner

4

Backup dos arquivos

Realizar o backup de todas as pastas e arquivos que for necessário nas pastas dos usuários e outras possíveis na unidade C:\

Manutenção

1

Análise e verificação de disco rígido

Analisar o disco com Western Digital Data Lifeguard Diagnostics e verificar com CHKDSK /f /r

2

Analise e verificação da instalação do Windows

Utilizar os comandos SFC /scannow e DISM /Online /Cleanup-Image /RestoreHealth para verificar a instalação do Windows

3

Limpeza de arquivos temporários e de instalação do Windows

No disco de instalação do Windows, propriedades do disco e limpeza de disco. 

4

Limpeza e desfragmentação avançada do disco.

Utilizar os programas cCleaner e Defraggler para manutenção avançada.

Instalação do Microsoft Windows

1

Instalação do Windows (7, 8 ou 10)

Computadores com até 3gb de memória ou processadores antigos, instalar x86. Computadores com 4gb e processadores mais novos instalar x64.

2

Instalação dos Drivers de Dispositivos

Verificar os drivers instalados e os que ficaram pendentes no Gerenciador de Dispositivos. Alguns drivers são instalados após o Windows Update

3

Atualizações do Windows Update

Tão logo termine de instalar os drivers realizar o Update do Windows para verificar atualizações do sistema e drivers.

4

Teste de memória

Realizar o Diagnóstico de Memória do Windows.

Instalação de Softwares Básicos

1

Pacote do Microsoft Office

Instalar componentes minimos para uso e os programas Word, Excel e Power Point. A excessão de clientes comerciais, neste caso pacote completo. Instalar

2

Compactador de Arquivos

Preferencialmente instalar o compactador 7zip. Para clientes comerciais instalar o Winrar.

3

Visualizador Multimídia

Instalar e configurar o VLC como player padrão.

4

Navegador

Preferencialmente instalar o Edge Chromium, mas dependendo do usuário instalar o Google Chrome

Instalação de Softwares de Segurança

1

Instalar software de antivírus

Caso o computador possua condições, instalar o Kaspersky Free. Caso não seja possivel, configurar adequadamente o Windows Defender.

2

Instalar software de anti-malwares

Fazer a instalação e atualização do Malwarebytes AdwCleaner

3

Instalar o uBlock Origin

Fazer o Download do bloqueador de conteudos.

Procedimentos Finais

1

Restaurar o backup

Salvar na área de trabalho em uma pasta com o nome do usuário, todos os backups realizados na mesma estrutura que se encontravam.

2

Verificação de vírus e malwares

Realizar uma verificação completa com o antivírus e malwares

3

Reiniciar e Desligar

Realizar o reinicio do computador varias vezes para verificar instalações e configurações adicionais, bem como desligar o computador para mesma finalidade.

4

Finalizar

Finalizar a OS, limpar o equipamento e embalar caso necessário.
